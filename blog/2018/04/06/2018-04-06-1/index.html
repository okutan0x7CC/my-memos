[CSS] Make hamburger menu with only css

  • CSS

2018-04-06

はじめに

ハンバーガーメニューをJavaScript等を使わず、CSSのみでシンプルに作りました。

ちなみに、このWebサイトの画面を狭めると出てくるのは、このハンバーガーメニューです。


コードはこちら


<!DOCTYPE html> 
<html>
    <head>
        <style type="text/css">
            .hamburger-menu #hamburger-handler {
                display: none;
            }
            .hamburger-menu .content {
                display: none;
            }
            .hamburger-menu .cross {
                display: none;
                left: 0px;
                top: 0px;
            }
            .hamburger-menu #hamburger-handler:checked ~ .content {
                display: list-item;
            }
            .hamburger-menu #hamburger-handler:checked ~ .cross {
                display: block;
            }
            .hamburger-menu #hamburger-handler:checked ~ .hamburger {
                display: none;
            }
            .hamburger-menu ul {
                list-style: none;
                padding-left: 0px;
            }
        </style>
    </head>
    <body>
        <div class="hamburger-menu">
            <input id="hamburger-handler" type="checkbox">
            <label class="hamburger" for="hamburger-handler">=</label>
            <label class="cross" for="hamburger-handler">×</label>
            <ul class="content">
                <a href="">
                    <li>aaa</li>
                </a>
                <a href="">
                    <li>bbb</li>
                </a>
                <a href="">
                    <li>ccc</li>
                </a>
            </ul>
        </div>
    </body>
</html>
